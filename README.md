# Violino

# Requirements

    - Node.js
    - Ruby
    - bundle (gem)
    - browserify (npm)
    - jshint (npm)
    - mocha (npm)
    - http-server (npm)
    - fswatch (brew)

# Getting started

Run a bundle install:

    $ bundle install
    $ npm install

Run the app locally:

    $ make build
    $ http-server dist -p 3000

# Running tests

    $ make mocha
